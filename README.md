A GitHub CLI extension to clone all repositories in an organization as quickly as possible.

## Prerequisites

- GitHub CLI version 1.0.0 or higher
- Git installed on your machine

## Installation

```sh
gh extension install svg153/gh-clone-org
```

## Usage

```sh
gh clone-org <org>
```

For help, run:

```sh
gh clone-org --help
```
